Horrid Green

'How horrid it must be
to be the color so green,
never thought to be loved,
never cried to be seen.

To lie upon a forest bed,
yet whine about the color so red.

Green blades dull and dancing in flow,
oh, the broken birch, pleading they go.

The calm passes,
winds have blown,
the mass gasps,
another color was born.

A new hue rose,
as dull as ever,
some beauty fades,
but memory is never.'
